# 启动逻辑

## 1. 运行前提

当前主线是纯 Python 实现，建议环境如下：

- Python 3.10+
- `numpy`
- `scipy` 可选

不需要：

- CMake
- Eigen
- C++ 编译器

## 2. 可执行入口

仓库根目录有两个脚本入口：

- `pinchRatioClustering.py`
- `genSimMatrix.py`

安装为包后也可以直接使用控制台脚本：

- `pinchRatioClustering`
- `genSimMatrix`

## 3. `pinchRatioClustering.py` 的主流程

执行顺序如下：

1. 解析命令行参数
2. 组装运行配置、数据配置和算法策略
3. 读取输入数据
4. 如果输入是点数据，则先构造相似度矩阵
5. 初始化顶点顺序
6. 运行 TILO
7. 按 PRC 指标选择切分点并递归划分
8. 输出标签、顺序和可选统计信息

## 4. 输入类型

### 4.1 点数据

使用：

- `--fileType 0`

特点：

- 每行一个样本
- 会先通过 Gaussian 或 KNN 构图，再进入聚类

### 4.2 邻接数据

使用：

- `--fileType 1`

特点：

- 直接读入图结构
- 可通过 `--adjNodeOffset` 兼容 0/1 起始编号

## 5. 基本命令

### 5.1 邻接图

```powershell
python pinchRatioClustering.py --dataInput tests/d1.txt --fileType 1 --adjNodeOffset 0 --useSparseMatrix 1 --numpart 2
```

### 5.2 点数据

```powershell
python pinchRatioClustering.py --dataInput tests/iris_all.txt --fileType 0 --tagLoc 1 --pointSimilarity 2 --knnAdjMode 3 --knnAdjK -1 --numpart 3
```

## 6. 输出文件

程序可生成两类主要输出：

- 标签文件
- 顺序/边界文件

默认命名规则：

- 标签：`<dataFile><outputLabelSuffix><numPart><infoSuffix>[seed]`
- 顺序：`<dataFile><outputOrderSuffix><numPart><infoSuffix>[seed]`

默认后缀：

- `outputLabelSuffix = _part_`
- `outputOrderSuffix = _tilo_`

如果开启 `--saveMultipleRuns 1`，还会额外保存多次初始化的结果信息。

## 7. 参数分组

### 7.1 运行参数

- `--seed`
- `--numpart`
- `--verbose`
- `--saveOrder`
- `--saveLabels`
- `--seedSuffixFlag`
- `--outputLabelSuffix`
- `--outputOrderSuffix`
- `--infoSuffix`
- `--initOrderFile`
- `--initLabelsFile`
- `--numInitOrderings`
- `--saveMultipleRuns`

### 7.2 数据参数

- `--dataInput`
- `--fileType`
- `--useSparseMatrix`
- `--commaSeparated`
- `--adjNodeOffset`
- `--tagLoc`
- `--pointSimilarity`
- `--commentDelimiter`
- `--commentDelimeter`

### 7.3 PRC/TILO 策略参数

- `--tiloMaxIteration`
- `--tiloEpsilon`
- `--policyPRCRecurseTILO`
- `--policyReverseOrderOnSplit`
- `--policyReturnRecursiveOrder`
- `--policyPRCMetric`
- `--policyRefineTILO`
- `--policyEvalAllMetrics`
- `--policyReturnMetrics`

### 7.4 构图参数

KNN：

- `--knnAdjMode`
- `--knnAdjK`
- `--knnAdjSigma`

Gaussian：

- `--gausssimAdjMode`
- `--gausssimAdjSigma`
- `--gausssimAdjThreshold`

## 8. `genSimMatrix.py`

用途是把点数据转换为相似度矩阵文件：

```powershell
python genSimMatrix.py tests/iris_all.txt tests/iris_sim_out.txt --fileType 0 --tagLoc 1
```

它会复用一部分参数解析逻辑，但不会执行聚类切分。

## 9. 首要排错顺序

1. 输入文件是否存在
2. `fileType` 是否与数据格式匹配
3. `numpart`、`knnAdjK`、`tagLoc` 是否越界
4. 是否误用了未完成选项，例如 `useMultiLevelPRC`
5. Python 环境是否具备 `numpy`
