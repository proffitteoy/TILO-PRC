# 代码组织

当前仓库已经整理为"纯 Python 主线仓库"，并按模块化结构重构。

## 顶层结构

```text
prc_v0.1.05_d2013_04_30/
├─ pyprc/                        Python 核心实现包
├─ tests/                        样例数据与基础测试
├─ doc/                          中文文档与论文 PDF
├─ pinchRatioClustering.py       CLI 入口脚本
├─ genSimMatrix.py               CLI 入口脚本
├─ prc.py                        旧导入路径兼容入口
├─ pyproject.toml                Python 打包配置
├─ README.md                     项目说明
└─ COPYING.GPL                   许可证
```

## `pyprc/` 模块结构

核心实现已从单文件 `core.py` 拆分为以下模块：

| 模块 | 职责 |
|------|------|
| `enums.py` | 枚举类型（KNNAdjMode、PrcMetricEnum 等）、FlatStruct、PRCError |
| `structs.py` | 所有策略/配置/返回值数据类 |
| `rng.py` | C++ 兼容确定性随机数生成器（MSVC LCG、MT19937） |
| `matrix.py` | BoundaryObject、SparseAdjacencyData、MatrixStorage |
| `order.py` | OrderObject（节点排列）、VirtualOrderObject（子排列视图） |
| `similarity.py` | 高斯/KNN 相似度矩阵构建（稠密和稀疏） |
| `io.py` | 文本矩阵、METIS 图文件读取、calcPurity |
| `algorithm.py` | 边界计算、TILO 位移、切分点查找、PRC 递归二分 |
| `cli.py` | 命令行参数解析、readData、两个 CLI 入口函数 |
| `compat.py` | 旧版 C++ 扩展兼容函数（已标记弃用） |
| `core.py` | 重导出层，保持 `from pyprc.core import X` 的旧路径兼容 |
| `__init__.py` | 公开 API 导出 |
| `cli_pinch_ratio_clustering.py` | 控制台脚本入口 |
| `cli_gen_sim_matrix.py` | 控制台脚本入口 |

依赖关系严格单向，无循环导入：

```
enums（无依赖）
├── structs
├── rng（无依赖）
├── matrix → order / similarity / io
├── algorithm（依赖 enums, structs, rng, matrix, order）
├── cli（依赖所有模块）
└── compat（依赖 algorithm, order, matrix）
```

## 根目录脚本

- `pinchRatioClustering.py` — 直接转发到 `pyprc` 的主 CLI
- `genSimMatrix.py` — 直接转发到 `pyprc` 的相似度矩阵生成 CLI
- `prc.py` — 为旧代码保留的兼容导入入口

## `tests/`

- 样例数据：`d1.txt`、`d1_initialorder.txt`、`iris_all.txt`
- 基础测试：`test_pyprc_smoke.py`

## `doc/`

中文文档目录，包含论文背景、运行说明、模块拆分说明和原始论文 PDF。
